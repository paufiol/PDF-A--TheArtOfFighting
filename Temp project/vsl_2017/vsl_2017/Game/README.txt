ART OF FIGHTING
__________________________________________________
2D fighting game, 1992 arcade-released game by SNK.
This specific version is a prototype made by PuñoEterno in C++
and using SDL2 libraries.

How to play
___________
Your goal is to either defeat the enemy or be the healthiest player 
when the round time is up.

Controls
________
SPACE as enter Key (To enter game, select character etc.)

PLAYER 1:
A/D: Forward/Backward
W: Jump
S: Crouch
Q: Punch
E: Kick
S, D, Q, E: Kouken

PLAYER 2:
J/L: Forward/Backward
I: Jump
K: Crouch
O: Kick
K, L, U, O: Kouken

DEBUG:
F1: Show/hide hitboxes
F2: Instant win P1
F3: Instant win P2
F4: God Mode

F6: Automatic Kouken
